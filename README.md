### Blockly VN32X

This tool allows anyone to make a [visual novel][] for the [Sega 32X][] using [Blockly][]  plus [electron][]

![blockly][Blockly image]

Right now, it's a work in progress.

Based on [BlocklyDuino][].

Uses [vn32x][] as its backend.
